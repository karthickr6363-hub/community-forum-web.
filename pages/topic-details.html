Getting Started with React Hooks

I'm new to React and want to understand how hooks work. Can someone explain the basics and provide some examples?

I've been learning React for a few weeks now, and I keep hearing about "hooks" but I'm not entirely sure what they are or when to use them. From what I understand, they let you use state and other React features in functional components.

Could someone provide:

  • A simple explanation of what hooks are
  • Common hooks and their use cases
  • Code examples showing how to use them
  • Best practices and common pitfalls

Thanks in advance for your help!

BEST ANSWER

Great question! React Hooks are functions that let you "hook into" React state and lifecycle features from function components.

Here are the most common hooks:

import { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);
  
  useEffect(() => {
    document.title = `You clicked ${count} times`;
  });
  
  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}

useState: Manages component state

useEffect: Handles side effects (like API calls)

useContext: Accesses React context

Adding to Sarah's great answer, here are some best practices:

  • Only call hooks at the top level - not inside loops or conditions
  • Only call hooks from React functions
  • Use the exhaustive-deps rule in ESLint

Common mistake: Forgetting dependencies in useEffect can cause bugs!
